# Shopee Affiliate Redirect System

System redirect yang memastikan user melewati link affiliate Shopee sebelum ke tujuan akhir (videy.co).

## Fitur
✅ Redirect ke affiliate Shopee terlebih dahulu  
✅ Auto-redirect ke videy.co setelah back  
✅ Rotasi link affiliate (round-robin)  
✅ Dashboard admin real-time  
✅ Analytics & tracking  
✅ Multi-device support  
✅ Fallback system
